# INFO 290: Fundamentals of Generative AI — Final Project Showcase

**Spring 2026 · UC Berkeley School of Information**

A static website showcasing the final projects from INFO 290, presented at the Spring 2026 showcase event on **April 30, 2026** at South Hall, UC Berkeley.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Landing page with event details and stats |

## Usage

Open `index.html` in a browser — no build step or server required. All styles are inlined and the project data is embedded as a JavaScript array in `projects.html`.
